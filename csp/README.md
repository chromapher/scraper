# Script tự tra điểm thi thử chuyên sư phạm
## Cách sử dụng
1. Đặt giá trị các biến sau:

``` python
subject = "{tên của môn}" # tên của môn
start = {số bắt đầu}
end = {số kết thúc}
```

Ví dụ:
```python
subject = "CS" # chuyên sinh
start = 1579 # băt đầu tra từ 1579
end = 1811 # kết thúc tra ở 1811      
```


2. Chạy script sử dụng 

```bash python main.py ```
